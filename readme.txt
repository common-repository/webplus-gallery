== Description ==

Creating a WordPress gallery is quick and easy.
Use the simple and convenient WebPlus Gallery plugin to create a gallery in minutes.
The WebPlus Gallery plugin allows you to create galleries with slide effects.
Using a block in Gutenberg editor, you can preview the image gallery before publishing it on your site.

We believe that the plugin "WebPlus Gallery" has minimal, but necessary functionality to create a gallery of slides.

== Installation ==

1. Upload `webplus-gallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the "WebPlus Gallery" section in your WordPress admin area.

== Frequently Asked Questions ==

= Does Gutenberg Editor have a plugin block =
In the Gutenberg editor, there is a plugin block called "WebPlus Gallery". It allows you to quickly insert a gallery into a post, as well as choose the type of gallery display.

= Can I use a shortcode =
When you create a gallery in the gallery manager, it has its own shortcode that you can use when publishing the gallery.

= Are images compressed on upload =
When images are loaded, they are automatically compressed to the correct size.

= Can I enter text in the "alt" tag =
Yes, in the gallery manager, when adding images, you can write them.

= How the gallery is displayed on mobile devices =
The gallery displays perfectly on mobile devices.

= Is there a lightbox in the gallery =
The PRO version has a lightbox to enlarge images from the gallery. You can order the pro version by writing to the mail: pavel.borysenko@gmail.com
